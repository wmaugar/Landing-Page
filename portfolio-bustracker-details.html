Descripción del proyecto

Este proyecto utiliza Mapbox y MBTA que son API's (Interfaz de Programación de Aplicaciones). Mapbox nos permite utilizar mapas y coordenadas georeferenciadas para ubicar objetos, y MBTA es una API de un sistema de transporte en Boston que proporciona información en tiempo real de sus buses.

La información obtenida es procesada, y se presenta al usuario en el documento, indicando la posición en tiempo real de un bus.
